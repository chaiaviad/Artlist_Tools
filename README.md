# Artlist Tools

דף נחיתה פנימי שמפצל לכלים של Artlist. הכל רץ בדפדפן, ללא שרת.
מיועד לפריסה כ-GitHub Pages תחת repo בשם `Artlist_Tools`.

## מבנה
- `index.html` — דף הנחיתה (hub) → `/Artlist_Tools/`
- `Collab_Maker/index.html` → `/Artlist_Tools/Collab_Maker/`
- `Thumbnail_Banner_Maker/index.html` → `/Artlist_Tools/Thumbnail_Banner_Maker/`
- `Guide/` — נכסי הגייד הרשמי

## פריסה (GitHub Pages)
1. צור ב-GitHub repo ריק בשם `Artlist_Tools` (תחת המשתמש chaiaviad), בלי README.
2. מהתיקייה הזו:
   git remote add origin https://github.com/chaiaviad/Artlist_Tools.git
   git branch -M main
   git push -u origin main
3. Settings → Pages → Branch: main / root → Save.
